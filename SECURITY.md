# Security Policy

## Supported versions

Only the latest [release](https://github.com/Alex123666tw/garbro-cli/releases) is supported.

## Reporting a vulnerability

Please report security issues **privately**:

1. Go to the repository's **Security** tab → **Report a vulnerability** (GitHub private advisories).
2. Do **not** open a public issue for security problems.

Include: a description of the issue, steps to reproduce, the affected version, and the potential impact. A typical concern for an extraction tool is a malformed archive that triggers unsafe behaviour (path traversal, resource exhaustion, crashes) — please attach a minimal sample if you can share it.

We will acknowledge your report and keep you updated on the fix.
